CSS属性

# 属性(Properties)

# 定位(Positioning)

参考：定位 Positioning (opens new window)

属性 版本 继承性 描述
position (opens new window) CSS2/3 无 用于指定一个元素在文档中的定位方式
z-index (opens new window) CSS2 无 定义一个元素在文档中的层叠顺序
top (opens new window) CSS2 无 定义了元素的上外边距边界与其包含块上边界之间的偏移
right (opens new window) CSS2 无 定义了元素的右外边距边界与其包含块右边界之间的偏移
bottom (opens new window) CSS2 无 定义了元素的底外边距边界与其包含块底边界之间的偏移
left (opens new window) CSS2 无 定义了元素的左外边距边界与其包含块左边界之间的偏移
clip (opens new window) CSS2/3 无 定义了元素的哪一部分是可见的。区域外的部分是透明的

# 布局(Layout)

参考：布局 Layout (opens new window)

属性 版本 继承性 描述
display (opens new window) CSS2/3 无 定义了元素是否显示，及生成哪种盒用于显示
float (opens new window) CSS1 无 定义了元素向左或者向右浮动放置。请参阅clear (opens new window)属性
clear (opens new window) CSS1 无 定义了一个元素是否可以放置在它之前的浮动元素旁边，或者必须向下移动在新行中放置。请参阅float (opens new window)属性
visibility (opens new window) CSS2 有 定义了元素是否可见。与display (opens new window)属性不同，此属性为隐藏的对象保留其占据的物理空间
overflow (opens new window) CSS2/3 无 简写属性。定义了元素处理溢出内容的方式

# 尺寸与补白(Dimension)

参考：尺寸 Dimension (opens new window)

参考：外边距 Margin (opens new window)

参考：内边距 Padding (opens new window)

属性 版本 继承性 描述
width (opens new window) CSS1 无 定义了元素内容区（Content Area）的宽度
min-width (opens new window) CSS2 无 定义了元素内容区（Content Area）的最小宽度
max-width (opens new window) CSS2 无 定义了元素内容区（Content Area）的最大宽度
height (opens new window) CSS1 无 定义了元素内容区（Content Area）的高度
min-height (opens new window) CSS2 无 定义了元素内容区（Content Area）的最小高度
max-height (opens new window) CSS2 无 定义了元素内容区（Content Area）的最大高度
margin (opens new window) CSS1 无 为元素设置所有四个方向（上右下左）的外边距
padding (opens new window) CSS1 无 为元素设置所有四个方向（上右下左）的内边距，即内容和元素边界之间的空间

# 背景与边框(Backgrounds and Borders)

参考：背景 Background (opens new window)

参考：边框 Border (opens new window)

属性 版本 继承性 描述
border (opens new window) CSS1 无 简写属性。定义元素边框的外观特性。参阅outline (opens new window)属性
border-width (opens new window) CSS1 无 简写属性。定义元素的边框厚度
border-style (opens new window) CSS1 无 简写属性。定义元素的边框样式
border-color (opens new window) CSS1 无 简写属性。定义元素的边框颜色
box-shadow (opens new window) CSS3 无 定义元素的阴影
border-radius (opens new window) CSS3 无 简写属性。定义元素的圆角
border-image (opens new window) CSS3 无 简写属性。定义将图像应用到元素的边框上
border-image-source (opens new window) CSS3 无 定义元素边框样式所使用的图像。
border-image-slice (opens new window) CSS3 无 用以指定从哪 4 个位置分割图像（遵循上右下左的顺序）
border-image-width (opens new window) CSS3 无 定义元素边框图像的厚度
border-image-outset (opens new window) CSS3 无 定义边框图像从边框边界向外偏移的距离
border-image-repeat (opens new window) CSS3 无 定义分割图像怎样填充边框图像区域
background (opens new window) CSS1/3 无 简写属性。定义元素的背景特性
background-color (opens new window) CSS1 无 定义元素使用的背景颜色
background-image (opens new window) CSS1/3 无 定义元素使用的背景图像
background-repeat (opens new window) CSS1/3 无 定义元素的背景图像如何填充
background-attachment (opens new window) CSS1/3 无 定义滚动时背景图像相对于谁固定
background-position (opens new window) CSS1/3 无 指定背景图像在元素中出现的位置
background-origin (opens new window) CSS3 无 指定的背景图像计算background-position (opens new window)时的参考原点(位置)
background-clip (opens new window) CSS3 无 指定对象的背景图像向外裁剪的区域
background-size (opens new window) CSS3 无 定义背景图像的尺寸大小

opacity (opens new window) CSS3 无 检索或设置对象的不透明度

参考：字体 Font (opens new window)

属性 版本 继承性 描述

font-style (opens new window) CSS1 有 指定元素的文本是否为斜体
font-variant (opens new window) CSS1 有 定义元素的文本是否为小型的大写字母
font-weight (opens new window) CSS1 有 定义元素文本字体的粗细
font-size (opens new window) CSS1 有 定义元素的字体大小
font-family (opens new window) CSS1 有 定义元素文本的字体名称序列
font-stretch (opens new window) CSS3 有 定义元素的文字是否横向拉伸变形
font-size-adjust (opens new window) CSS3 有 定义小写字母 x 的高度与对象文字字号的比率。

# 文本(Text)

参考：文本 Text (opens new window)

属性 版本 继承性 描述
text-transform (opens new window) CSS1/3 有 定义元素的文本如何转换大小写
white-space (opens new window) CSS1 有 指定元素是否保留文本间的空格、换行；指定文本超过边界时是否换行。
tab-size (opens new window) CSS3 有 定义元素内容中制表符的长度
word-break (opens new window) CSS3 有 定义元素内容文本的字间与字符间的换行行为
word-wrap/overflow-wrap (opens new window) CSS3 有 定义元素内容文本遇到边界时如何换行
text-align (opens new window) CSS1/3 有 定义元素内容的水平对齐方式
text-align-last (opens new window) CSS3 有 定义块内文本内容的最后一行（包括块内仅有一行文本的情况，这时既是第一行也是最后一行）或者被强制打断的行的对齐方式
text-justify (opens new window) CSS3 有 定义使用什么方式实现文本内容两端对齐
word-spacing (opens new window) CSS1/3 有 指定单词之间的额外间隙
letter-spacing (opens new window) CSS1/3 有 指定字符之间的额外间隙
text-indent (opens new window) CSS1/3 有 定义块内文本内容的缩进
vertical-align (opens new window) CSS1/2 无 定义行内元素在行框内的垂直对齐方式
line-height (opens new window) CSS1 有 定义元素中行框的最小高度
text-size-adjust (opens new window) CSS3 有 定义移动端页面中元素文本的大小如何调整

# 媒体查询(Media Queries)

参考：媒体查询 Media Queries (opens new window)

# 定义和用法

<media_query_list>：[<media_query>[','<media_query> ]*]?

<media_query>：[only | not]? <mediaquery> [and <expression>] | <expression> [and <expression>]_

<expression>：'('<media_feature>[:<value>]?')'

# 使用说明

通过不同的媒体类型和条件定义样式表规则。

  • 媒体查询让 CSS 可以更精确作用于不同的媒体类型和同一媒体的不同条件。

  • 媒体查询的大部分媒体特性都接受 min 和 max 用于表达“大于或等于”和“小与或等于”。如：width 会有 min-width 和 max-width

  • 媒体查询可以被用在 CSS 中的@media 和@import 规则上，也可以被用在 HTML 和 XML 中。

    示例代码：

    @media screen and (width:800px){ … }
@import url(example.css) screen and (width:800px);
<link media="screen and (width:800px)" rel="stylesheet" href="example.css" />
<?xml-stylesheet media="screen and (width:800px)" rel="stylesheet" href="example.css" ?>

# 媒体特性 Media Features

媒体特性 取值 接受 min/max 描述
width <length> yes 定义输出设备中的页面可见区域宽度
height <length> yes 定义输出设备中的页面可见区域高度
device-width <length> yes 定义输出设备的屏幕可见宽度
device-height <length> yes 定义输出设备的屏幕可见高度
orientation portrait | landscape no 定义'height'是否大于或等于'width'。值 portrait 代表是，landscape 代表否
aspect-ratio <ratio> yes 定义'width'与'height'的比率
device-aspect-ratio <ratio> yes 定义'device-width'与'device-height'的比率。如常见的显示器比率：4/3, 16/9, 16/10
color <integer> yes 定义每一组输出设备的彩色原件个数。如果不是彩色设备，则值等于 0
color-index <integer> yes 定义在输出设备的彩色查询表中的条目数。如果没有使用彩色查询表，则值等于 0
monochrome <integer> yes 定义在一个单色框架缓冲区中每像素包含的单色原件个数。如果不是单色设备，则值等于 0
resolution <resolution> yes 定义设备的分辨率。如：96dpi, 300dpi, 118dpcm
scan progressive | interlace no 定义电视类设备的扫描工序
grid <integer> no 用来查询输出设备是否使用栅格或点阵。只有 1 和 0 才是有效值，1 代表是，0 代表否
