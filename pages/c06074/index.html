块元素列表

# 块状元素列表

https://developer.mozilla.org/zh-CN/docs/Web/HTML/Block-level_elements

# 定义

  1. 元素前后带有换行符，此元素总是单独占据一行；
  2. 宽度缺省是它的容器的 100%；
  3. 可以设置width 和 height属性;(注意：块级元素即使设置了宽度，仍然是独占一行的)；
  4. 可以设置margin 和 padding；
  5. 块级元素只能出现在 body 元素内；
  6. 一般块级元素可以包含行内元素和其他块级元素。这种结构上的包含继承区别可以使块级元素创建比行内元素更”大型“的结构。

# 列表

  1. # address (opens new window)

    定义地址

  2. # article (opens new window)

    文章内容 （HTML5）

  3. # aside (opens new window)

    伴随内容（HTML5）

  4. # audio (opens new window)

    音频播放（HTML5）

  5. # blockquote (opens new window)

    块引用

  6. # canvas (opens new window)

    绘制图形

  7. # dl (opens new window)

    dl (opens new window)：定义列表

    dd (opens new window)：定义列表中定义条目描述

    dt (opens new window)：定义列表中的项目

  8. # div (opens new window)

    定义文档中的分区或节

    常用块级容易，也是 css layout 的主要标签

  9. # form (opens new window)

    form (opens new window)：创建 HTML 表单，交互表单

    fieldset (opens new window)：定义一个框架集，form 控制组

    legend (opens new window)： 元素为 fieldset 元素定义标题

  10. # figure (opens new window)

figure (opens new window)：图文信息组（HTML5），参照 figcaption

figcaption (opens new window)：图文信息组标题（HTML5）

  1. # footer (opens new window)

    区段尾或页尾（HTML5）

  2. # h1~6 (opens new window)

    大标题，副标题，3 级标题...，6 级标题

  3. # header (opens new window)

    区段头或页头

  4. # hgroup (opens new window)

    标题组

  5. # hr (opens new window)

    水平分隔线

  6. # nav (opens new window)

  7. # noscript (opens new window)

    定义在脚本未被执行时的替代内容

  8. # ol (opens new window)

    定义有序列表

    li：定义列表项目

  9. # output (opens new window)

    表单输出（HTML5）

  10. # p (opens new window)

    定义段落

  11. # pre (opens new window)

    定义预格式化的文本

  12. # section (opens new window)

    一个页面片段

  13. # table (opens new window)

    table (opens new window)：表格

    tbody (opens new window)：表体

    thead (opens new window)：表头

    td (opens new window)：表格中的标准单元格

    th (opens new window)： 定义表头单元格

    [tr]： 定义表格中的行

    tfoot (opens new window)：表脚注

  14. # ul (opens new window)

    定义无序列表

  15. # video (opens new window)

    视频（HTML5）

# 废弃

  1. # frame (opens new window)

    frame (opens new window)：窗口（框架）

    frameset (opens new window) ：包含 frame 的 HTML 元素

    noframes (opens new window)：为那些不支持框架的浏览器显示文本，于 frameset 元素内部

  2. # menu (opens new window)

    menu (opens new window)：（右键）菜单

    menuitem (opens new window)：菜单项
